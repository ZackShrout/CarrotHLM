◆ inverse() [1/2]

constexpr float4x4 chlm::inverse ( const float4x4 &  m)
constexprnoexcept

Computes the general inverse of a 4x4 matrix.

Uses Gauss-Jordan elimination with partial pivoting for numerical stability. This is a full inverse that works for any invertible 4x4 matrix (including those with projection, shear, or non-uniform scaling).

Note
This is significantly slower than affine_inverse() or assuming an orthonormal basis. Use only when a full inverse is required (e.g. for view-projection matrices or arbitrary transformations). For model matrices, prefer affine_inverse().
Parameters
mThe matrix to invert. Must be invertible.
Returns
The inverse matrix such that m * inverse(m) identity. If the matrix is singular (determinant 0), returns the identity matrix as a safe fallback.
